### Deploy :arrow_down:

#### [Front-Blog](https://front-blog-sepia.vercel.app/)


### Como Iniciar a aplicação na sua máquina :arrow_down:

```bash

# Clonar o repositório
$ git clone https://github.com/PedroFurlann/teste-front.git

# Entrar no diretório
$ cd teste-front

# Instalar as dependências
$ npm install

# Iniciar o projeto
$ npm run dev

```
